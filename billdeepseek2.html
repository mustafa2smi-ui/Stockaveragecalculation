❓ Help & FAQs

🚀 Quick Start Guide

  1. Tap 💸 Bill mode to start billing
  2. Search or tap an item to add it to the bill
  3. Use +/- buttons to adjust quantity
  4. Enter Customer Name in the header
  5. Press 💾 Local Save to store the bill
  6. Use 🖨️ Print for thermal printer

❓ Frequently Asked Questions

How do I add a new item?

Switch to 📝 Entry mode, fill in the item details (Name and Price are required), and press 💾 Save. The code will be auto-generated if left empty. Switch back to 💸 Bill mode to use the new item.

Can I change the price during billing?

Yes! When an item is in the bill, an orange rate input box appears. You can change the price directly there. This only affects the current bill, not the master item.

Does it work without internet?

Yes! All data is saved in your browser's Local Storage. The app works completely offline after the first visit. No internet required for daily billing.

How do I backup my data?

Go to 📝 Entry mode and use the 📤 Export button above the item list. Save the JSON file. To restore, use the 📥 Import button and select your backup file.

What does "Other" (₹) do?

It adds a miscellaneous charge to the bill - like packing charges, delivery fees, or any item not in your database. Enter the amount and press +.

Can I print bills?

Yes! Press 🖨️ Print to connect to a Bluetooth thermal printer (2-inch). The printer must support standard ESC/POS commands. Make sure Bluetooth is enabled on your device.

How do I edit a saved bill?

Switch to 📝 Entry mode to see saved bills. Tap ✏️ Edit on any bill to load it, make changes, and save again.
